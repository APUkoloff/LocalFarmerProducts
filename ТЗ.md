#Требования к веб-приложению Магазин локальных фермерских продуктов
##Общие требования:
1. Приложение должно иметь интуитивно понятный интерфейс, адаптивный дизайн для использования на всех типах устройств.
2. Возможность заполнения формы для совершения заказа.
3. Поддержка минимально необходимых версий браузеров (Chrome >=60, Firefox>=55, Edge>=79, Safari>=12)
##Функциональные требования:
1. Пользователь должен видеть список товаров (с указанием количества/веса, имеющегося в наличии), доступных для приобретения в его локации. Поддержка сортировки по категориям, названию, цене, количеству.
2. Формирование корзины (возможность добавить/убрать товар, изменить количество).
3. Редактирование профиля покупателя (изменение контактных данных, привязанных карт и адреса доставки)
4. Профиль продавца (контактные данные, адрес, счет для зачисления средств) с возможностью добавления, редактирования, удаления своих товаров.
5. Аналитика продаж и просмотр заказов (для администрации - фильтры за неделю/месяц/год).
6. Просмотр активных заказов для покупателя и продавца.
##Другие функциональные требования (необязательно):
1. Репутация пользователя.
